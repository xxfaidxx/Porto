Projects

Project Terbaru

Berikut beberapa project yang pernah saya kerjakan menggunakan teknologi web modern.

DiagnoSmart

DiagnoSmart adalah platform diagnosis cerdas berbasis web yang dikembangkan sebagai proyek akhir dari Coding Camp Dicoding bekerja sama dengan DBS Foundation. Tujuan utama proyek ini adalah untuk membantu pengguna mengenali potensi penyakit sejak dini melalui teknologi modern.

Notes Application

Proyek Notes Application adalah aplikasi web untuk mencatat dan mengelola kosakata dalam berbagai bahasa menggunakan metode flashcard. Aplikasi ini dirancang untuk membantu pengguna dalam proses pembelajaran bahasa dengan cara yang interaktif dan efisien.

Bintang Cendekia

Project Bintang Cendekia merupakan aplikasi berbasis web yang dirancang untuk sistem Penerimaan Peserta Didik Baru (PPDB). Aplikasi ini menyediakan informasi tentang sekolah, seperti nama, alamat, dan fasilitas yang tersedia, serta memungkinkan calon siswa untuk mendaftar dengan mengisi data pribadi dan mengunggah dokumen yang dibutuhkan. Selain itu, pihak sekolah dapat memverifikasi dan mengelola data pendaftaran siswa baru, serta memantau status pendaftaran, apakah diterima, menunggu, atau ditolak.
